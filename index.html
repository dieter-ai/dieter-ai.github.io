Projects

OutBox

OutBox is a web application leveraging AI function calls to organize unstructured data. The project includes advanced extraction and processing of HTML content using Python packages such as BeautifulSoup, re, json, pandas, and scipy. It detects and clusters HTML elements containing prices, identifies the largest cluster, and determines the lowest common ancestor and its direct child elements within the identified clusters.

SolarSim

SolarSim is a Python-based tool that determines sunlight exposure at specific GPS coordinates, considering potential obstructions from buildings or landscapes. It utilizes the Digitaal Hoogtemodel Vlaanderen II (DSM) and a 1-meter resolution raster to calculate sun visibility accurately.

Pochemuchka

Pochemuchka is a Python application that uses open-source language models (LLMs) locally to interpret live webcam or camera feeds. It describes ongoing activities and provides real-time transcriptions (with an option for English translations) of spoken words, enhancing situational awareness and context understanding.
